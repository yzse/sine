### Visualizing the Sine wave using p5.js

"Art" work made using [p5.js](https://p5js.org/get-started/). 

A Sinusodial wave ("sine wave") as a function of distance is a single-frequency traveling wave. It is a one-dimension motion that wiggles over time is the mathematical basis to explain sounds, vibrations, pendulums... it makes circles "circular" in the same way lines make squares "square". Learn more about the sine wave [here](https://betterexplained.com/articles/intuitive-understanding-of-sine-waves/).
